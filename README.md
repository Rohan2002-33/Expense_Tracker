# CLI Expense Tracker

A command-line Dart application to add, view, and total daily expenses.

## Features
- Add Expense
- View All Expenses
- Show Total Expenses
- Exit Application

## OOP Concepts Demonstrated
- **Inheritance**: `FoodExpense`, `TransportExpense`, `EntertainmentExpense` extend base `Expense` class
- **Polymorphism**: `EntertainmentExpense` overrides `display()` method
- **Extension Method**: `AmountFormatting` extension on `double` formats currency
